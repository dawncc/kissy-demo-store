##项目前约定

- 使用clam自动构建（使用grunt自动构建项目目录、合并、压缩js和css文件、合并http请求）
- 遵守KISSY 模块定义规范（KMD）（http://docs.kissyui.com/1.4/docs/html/guideline/kmd.html）
- 遵守 注释规范

### 注释

模块注释

```javascript
/**
 * @fileoverview 文件用户
 * @author 作者的
 **/
```

函数注释

```javascript
  /**
     * @name ImageZoom
     * @class 图片放大器
     * @since 1.4
     * @constructor
     * @extends Base
     */
```

版本变更注释

```javascipt
/**
 * changes:
 * author: dawncc

 */
```
